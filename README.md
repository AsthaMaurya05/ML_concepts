# ML Concepts

This repository contains simple machine learning examples in Python.

## Files

- `linear_regression.py` - linear regression on the California Housing dataset
- `logistic_regression.py` - logistic regression on the Breast Cancer dataset
- `confusion_matrix.py` - confusion matrix example
- `decision_tree.py` - decision tree classification example
- `random_forest.py` - random forest classification example
- `knn.py` - k-nearest neighbors example
